分布特征

DBSCAN（Density-based Spatial Clustering of Application with Noise）是由Ester等人在1996年提出的一种基于密度的聚类算法，是M-DBSCAN的核心。该算法的核心是使空间中一定距离(Eps)范围内的点对象数目达到指定数目(MinPt)的所有对象成为一个独立的分类，该模型可以在含有噪声空间的数据中将以高密度的对象划分为一类，并且不受聚类类别数和形状的限制[10]。DBSCAN要求对于单个类中的每一个点，在给定范围内至少有一定数量的领域点[11]。

经过距离矩阵直方图绘制后发现，距离密度在区间0.00108-0.217之间呈现上升趋势，之后呈现下降趋势。在拐点进行聚类分析后，得到全县内为一大类，因此将区间缩小至中值，选取区间0.001-0.1进行DBSCAN聚类分析，结果如图(图 5)所示在这里，我们可以把聚类结果认为无差状态，即每个聚类结果单纯为一热点区域，而没有较大的属性差别。 Eps为0.007时出现了两点之间的聚类(a)，主要分布于县内各乡镇，说明各镇的廊桥一般不会单独出；Eps为0.03时出现了最小为5点之间聚类(b)，但类别数远低于前者，说明拥有5座桥以上的区县较少，主要为松源镇、岭头乡、举水乡、合湖乡、安南乡。Eps为0.05时，将聚集区划分为西北角，西南角，东南部三个主要部分，与地形差异契合，西南角以平原为主，西北角为进入庆元县中部平原的廊道，东南角以山地为主，而未被归入大类中的东北角以高山地形为主；Eps为0.08时，区分出了最小值为10的聚类情况，将类别进一步扩大化，大致可看为庆元县腹地的大聚集区，与龙泉县交界的黄田竹口聚集区，与景宁交界的百山祖合湖山地聚集区。

通过绘制标准差(Standard Deviational Ellipse,简称 SDE)椭圆来判断数据的离散程度，并能找到要素的离散趋势和扩散方向。如图所示，廊桥离散程度较高，扩展到接近整个县的范围，绘制的标准差椭圆大致接近县界线的形态，呈现东西走向。但椭圆中心偏向西部，即松源镇所在的地区。


摘录笔记

一些常用的html元素，特别是html文本元素的语义化使用中，也有一些可以优化及需要注意的地方

  • <strong>和<b>，<em>和 <i>，其效果差不多是一样的，但是前者的语义化更加友好白色空间折叠：当浏览器遇到两个或者两个以上的白色空间元素（空格，换行）时，会折叠成一个空格。自己编码时注意换行应该用 <br>。
  • 文档中的一些嵌入式内容，比如引用的图片，插图，表格，代码段等，可以作为独立的单元，当这部分转移到附录中或者其他页面时不会影响到主体，这样的元素都可以放在<figure>元素内，而且可以搭配其子元素<figcaption>作很好的元素说明或者备注信息
  • img元素最好附带alt信息，即对图片进行文本说明，当图像无法查看时会显示这段文本描述
  • table元素现在也有更好的语义化结构元素
    • caption 表格的标题
    • thead 适合放表格的表头行
    • tbody 表格的主体部分
    • tfoot 表格的脚注部分
  • form表单元素，
    • label标签，为input元素定义标注，改进了表单控件的可用性，当你点击到label标签时，会自动聚焦到对应控件上，label标签一般有两种用法
      1. label的for属性与控件的id对应，比如 
        <label for="username">请输入用户名: </label><input type"text"= id="username" name="username">
        label内嵌控件，比如 
        <label>请输入用户名<input type="text" id="username" name="username"></label>
      placeholder属性，其值会在输入字段为空时显示，并会在字段获得焦点时消失
    • 对于表单中的单选radio控件和复选checkbox控件以及下拉框select控件，可以为radio, checkbox添加checked属性以及为option添加selected属性让其默认选中
